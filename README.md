# 🧑‍🏫 Форма набора наставников для колледжа

Яркая поп-арт форма для сбора заявок от студентов 2-4 курсов, готовых помогать первокурсникам.

---

## ✨ Особенности

- 🎨 **Поп-арт дизайн** — яркие цвета, стикеры, анимированные руки
- 📱 **Адаптив** — отлично выглядит на телефонах и компьютерах
- 🤖 **Telegram интеграция** — заявки приходят в Telegram-бот
- 📎 **Загрузка файлов** — можно прикрепить резюме или фото
- ✅ **Валидация** — все обязательные поля проверяются

---

## 🚀 Как использовать

1. Откройте страницу: https://starpoint83.github.io/nastavniki/
2. Заполните все поля
3. Нажмите "Отправить заявку"
4. Данные придут в Telegram

---

## 📋 Поля формы

| Поле | Описание |
|------|----------|
| ФИО | Полное имя кандидата |
| Телефон | Контактный номер |
| Email | Электронная почта |
| Курс | 2-й, 3-й или 4-й курс |
| Специальность | Направление обучения |
| Группа первокурсников | С какой группой готов работать |
| Опыт наставничества | Был ли наставником раньше |
| Навыки | Чем может помочь (множественный выбор) |
| Мотивация | Почему хочет стать наставником |
| Дополнительно | Соцсети, портфолио |
| Файл | Резюме или фото (необязательно) |
